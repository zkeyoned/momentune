# Momentune — 项目介绍

> **Every moment has its own melody.**
> 每个瞬间，都有一首属于它的旋律。

---

## 一、项目概述

**Momentune**（Moment × Tune）是一款面向生活记录的 AI 驱动移动 Web 应用。用户在旅行、生活中拍摄照片的瞬间，AI 会自动分析照片的情绪与场景，把它映射到一个 **Valence–Arousal 情绪坐标** 上，再从坐标推荐最贴合此刻的音乐，并**在 App 内直接播放完整歌曲**，同时将这一「瞬间」保存为一页图文音乐日记。支持时间线和地图两种回顾模式，让回忆可听、可见、可回溯。

## 三、核心创意

> 到了景点，拍一张照片 → AI 读出它的情绪坐标 → 推荐一首最合适的歌 → 直接听 → 写下此刻感受 → 存为一页日记。

Momentune 的核心洞察是：**每一张有意义的照片背后，都有一种难以名状的情绪**。AI 通过图像分析帮你「翻译」这种情绪，并用音乐来「表达」它。多年后翻看时，照片唤起画面，音乐唤起感受，形成复合的回忆体验。

---

## 四、核心技术 · 情绪坐标映射

「AI 把视觉翻译成音乐」听起来很玄，Momentune 把它落成一个**可解释、可复现的系统**：不靠模型凭感觉乱配，而是让照片和音乐在同一个「情绪坐标系」里相遇。

这个坐标系借鉴心理学的**情绪二维模型（Valence–Arousal）**：横轴是**效价**（消极 ↔ 积极），纵轴是**唤醒度**（平静 ↔ 高能）。

它怎么运转：
1. **照片定位**：视觉模型分析照片的色温、明暗、饱和度、场景、时段、天气，输出一组 (效价, 唤醒度) 坐标值。
2. **音乐定位**：曲库里每首歌预先标好同一套 (效价, 唤醒度) 坐标。
3. **最近邻匹配**：在坐标系里找出离照片最近的几首歌，再叠加流派 / 年代偏好微调，推荐 3–5 首。

这套设计既**讲得清**（「凭什么配这首歌」有坐标可答），又**防得住**（是一套设计出来的系统，而不是一句能被抄走的 prompt），还正好把跨模态情绪映射这个最独特的技术创意摆到了台面上。

## 六、功能模块

### 1. 智能拍照
- 调用手机相机拍摄（或从相册选取）
- 支持实时预览 + 重新拍摄
- 自动获取地理位置信息

### 2. AI 图像分析（输出情绪坐标）
- 场景识别：自然风光、城市街景、室内、人物等
- 视觉特征提取：主色调、明暗、天气、时间段
- **输出 Valence–Arousal (效价, 唤醒度) 情绪坐标**，作为配乐的依据

### 3. 音乐推荐 + App 内整首播放
- 根据情绪坐标在曲库里做最近邻匹配，推荐 3–5 首
- **App 内整首播放**：
  - 接入用户自己的网易云 / QQ 账号（用户授权登录），App 借助开源接口（NeteaseCloudMusicApi）以用户身份取到完整歌曲地址，用 `<audio>` 直接在 App 内播放完整版
  - 本质是为用户的账号提供一个第三方播放客户端——能听什么、有没有完整版，取决于用户账号自身的会员与版权
  - **兜底**：少数 VIP / 下架歌曲取不到时，回退到内置免版权曲库，保证每张卡片都有声音
- 卡片展示歌曲封面、歌名、歌手 + 播放控制

### 4. 图文音乐日记
- 支持用户撰写文字感想
- 日记卡片展示：照片 + AI 情绪描述 + 文字感想 + 歌曲封面 + 播放按钮
- 一键保存，形成日记列表

### 5. 时间线回顾模式
- 按时间倒序展示所有日记
- 瀑布流 / 卡片式布局
- 点击卡片可回顾当天的照片、文字和歌曲

### 6. 地图回顾模式
- 基于地理位置在地图上标注所有记录点
- 点击标注弹出日记卡片预览
- 支持缩放浏览不同区域的记忆

### 7. 年度情绪轨迹（加分项）
- 把每篇日记的情绪坐标连成一条曲线，生成「年度情绪回顾」，让你看见自己这一年的情绪起伏

## 八、技术方案

| 层级 | 技术选型 |
|------|----------|
| **前端框架** | React + TypeScript（移动端 Web + PWA） |
| **后端服务** | Node 中间商（NeteaseCloudMusicApi），用户授权后以其身份取歌曲地址 |
| **移动端能力** | 浏览器相机 / 相册、Geolocation 定位 |
| **AI 图像分析** | 国内多模态视觉 API（Qwen-VL / GLM-4V / 豆包视觉），输出情绪坐标 |
| **情绪引擎** | Valence–Arousal 二维坐标 + 最近邻匹配 |
| **音乐来源** | 接入用户网易云 / QQ 账号取完整歌曲 + 免版权曲库（预标 V-A 坐标）兜底 |
| **音乐播放** | App 内直接流式播放完整歌曲（`<audio>`） |
| **地图展示** | 高德地图 / Mapbox |
| **数据存储** | 本地 IndexedDB（Dexie.js，Demo 阶段）；可扩展账号 + 云同步 |

## 九、音乐来源与版权说明

Momentune 不自建版权曲库，而是**接入用户自己的网易云 / QQ 音乐账号**：用户授权登录后，App 借助开源接口（NeteaseCloudMusicApi）以用户自己的身份获取并播放完整歌曲，本质是为用户的账号提供一个第三方播放客户端，播放内容与权限完全取决于用户账号自身的会员与版权。

> 注：该方案依赖非官方接口，需自行评估稳定性与合规；少数 VIP / 下架歌曲可能取不到，会自动回退到内置免版权曲库兜底。
