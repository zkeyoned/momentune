## 一、基础框架搭建

### 1.1 情绪标签系统
- 从 16 个基础情绪标签扩展到 **71 个细分标签**（基于 V-A 二维坐标）
- 新增音乐特有标签：Groove、Ethereal、Hype 等
- 标签密度检查：加权距离 > 0.08 防冲突
- 文件：[src/algorithm/config/emotionLabels.ts](src/algorithm/config/emotionLabels.ts)

### 1.2 风格/场景标签系统
- 风格标签 **60 个**：rap / electronic / rnb / guofeng / kpop / country / soul / folk / rock / trap / dance 等
- 场景标签 **77 个**：city_night / night_drive / seaside_dusk / gaming / workout_cardio / rainy_window 等
- 场景×情绪先验适配矩阵

### 1.3 照片 → V-A 映射
- 文件：[src/algorithm/photoToVA.ts](src/algorithm/photoToVA.ts)
- 8 维特征融合：色调、亮度、饱和度、场景、时段、天气、人物、构图
- GPS 反向地理编码辅助（POI → 场景 → V-A）
- 深夜时段修正 + 置信度加权

### 1.4 音乐库三层架构
- 文件：[src/algorithm/musicLibrary.ts](src/algorithm/musicLibrary.ts)
- 热歌层（hot）：实时榜单歌曲
- 情绪层（emotion）：经典品质 + 用户导入
- 兜底层（fallback）：免版权纯音乐

## 二、匹配算法

### 2.1 五维加权匹配公式
- 文件：[src/algorithm/match.ts](src/algorithm/match.ts)

| 维度 | 最终权重 | 说明 |
|------|---------|------|
| scoreVA | 0.30 | V-A 坐标加权欧氏距离 |
| scoreScene | 0.10 | 场景标签匹配度 |
| scorePref | **0.35** | 用户偏好（风格+语言+偏好中心） |
| scoreSceneFit | 0.08 | 场景历史接受率 |
| scoreRefSim | 0.05 | 参考歌相似度 |
| scoreHot | 0.12 | 热歌新鲜度 |

### 2.2 偏好评分内部权重
- 风格匹配：**50%**（用户明确表达的就是风格偏好）
- V-A 偏好接近度：25%
- 语言匹配：15%
- 平台匹配：10%

### 2.3 两阶段推荐
- 文件：[src/algorithm/recommend.ts](src/algorithm/recommend.ts)
- 阶段 1：核心 8 首（情绪多样 + 风格覆盖 + 热歌保底）
- 阶段 2：扩展 7-12 首（全局多样化）
- 冷启动探索机制

## 三、关键参数调优

### 3.1 权重演进
- 文件：[src/algorithm/config/thresholds.ts](src/algorithm/config/thresholds.ts)

| 参数 | 最初值 | → | 最终值 | 原因 |
|------|--------|---|--------|------|
| 用户偏好权重 | 0.15 | → | **0.35** | 用户的选择应该有力量 |
| 冷启动偏好权重 | 0.08 | → | **0.22** | 新用户选了风格不该被忽略 |
| V-A 候选池阈值 | 0.45 | → | **0.60** | 风格多样的歌能进候选池 |
| 经典歌曲热歌分 | 0.50 | → | **0.65** | 经典不应该因"老"被惩罚 |
| 兜底层热歌分 | 0.30 | → | **0.45** | 保底音乐也不该被过度惩罚 |
| 深夜 V 值 | 0.32 | → | **0.42** | 深夜不只是忧伤，也可以有氛围 |
| 深夜修正幅度 | -0.08/-0.10 | → | **-0.03/-0.05** | 双重修正太狠（时段+修正） |
| 冷启动交互次数 | 5 | → | **3** | 更快进入正常模式 |

## 五、解决的问题记录

| 问题 | 根因 | 修复 |
|------|------|------|
| 推荐全是忧伤民谣 | 音乐库小 + 权重过分倾向 V-A | 扩充库 + 提升偏好权重到 0.35 |
| 用户选了风格感觉不到 | 冷启动偏好权重被降到 0.08 | 提升到 0.22 |
| 经典好歌永远排不上 | older 热歌分只有 0.5 | 提升到 0.65 |
| 深夜城市 = 忧伤是偏见 | night 的 V 固定为 0.32 | 调整到 0.42（多元场景） |
| 说唱/电子进不了候选池 | 候选池 V-A 阈值太严（0.45） | 放宽到 0.60 |
| 风格匹配影响太弱 | 偏好分里风格只占 30% | 提升到 50% |
